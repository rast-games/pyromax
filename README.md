# Pyromax

Асинхронный, модульный фреймворк для юзерботов в MAX Messenger.

## Возможности
- Асинхронное ядро.
- Dispatcher и Router.
- Фильтры и паттерны.
- Форматирование сообщений.
- Типизация основных объектов.

## Быстрый старт
Смотри полную документацию: [Pyromax Docs](./docs/index.md)

## Установка
```bash
uv add pyromax
```

## Разработка
```bash
uv add --dev mkdocs
uv run mkdocs serve
```
